By default, we assume that signtool.exe is in PATH. Here you can optionally specify the path to the directory in which signtool.exe is installed, excluding the architecture subdirectory.

For example, suppose that signtool.exe is located in:

C:\Program Files (x86)\Windows Kits\10\bin\10.0.18362.0\[arch]\signtool.exe

Then specify:

C:\Program Files (x86)\Windows Kits\10\bin\10.0.18362.0
