The digest algorithm(s) to use to creating signatures.

If not specified, uses Signtool's default (which is 'sha1' at the time of writing).

You can specify multiple ones by separating them with a space. Like this: 'sha256 sha1'

Recommendations

You should explicitly specify 'sha256', because Signtool's default — 'sha1' — is no longer considered secure.

If you need compatibility with older Windows machines that don't support 'sha256', then you can specify 'sha256 sha1'.
